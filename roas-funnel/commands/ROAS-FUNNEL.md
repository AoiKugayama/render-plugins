# ROAS-FUNNEL OS — ROASが絶対に合う導線設計

## 概要

「3ヶ月ROAS500%を達成する導線設計」を実現するための思考OS。
広告で取得したリストを最大限に活用し、長期にわたってROASを積み上げるための
フロント〜バックエンド構造・CS（カスタマーサポート）・メルマガ教育の3層設計を行う。

> **このOSが生み出す成果**
> - 初月ROAS100%未満でも、3ヶ月で500%、9ヶ月で700%に到達する構造
> - 同じ商品・同じオファーページのまま、教育を変えるだけで転換率+10%以上
> - CS導入でROASを1.5倍確実化（月100万円以下の投資で数千万円改善）

## フェーズ1：フロント導線設計

### フロント導線の3択

ヒアリング結果をもとに、以下の基準で最適な導線タイプを提案する。

| 導線タイプ | 向いている条件 |
|---|---|
| **電ク（1本動画 → 電話/Zoom クロージング）** | 最も汎用。個別相談でクロージングする場合の第1選択。商品単価が高い（30万〜）場合に特に有効 |
| **オートウェビナー** | コンセプト・素材が圧倒的に強い。個人ブランドが確立されている。認知度が高い |
| **3話型VSL** | ターゲットとの深いコミュニケーションが必要。情報リテラシーが高い層、または信頼形成に時間がかかる層を相手にする場合 |

### フロントが狙う意識層

```
超潜在層 ←──────────────────── → 超顕在層
   低関心                          購買直前

           ↑ここを狙う
        ミドル〜顕在層
   （認識はしているが比較・検討段階）
```

**フロントで達成する数値目標：ROAS 200%以上**

### 導線タイプ選定後の確認事項

```
【フロント導線確認シート】

Q-F1. 商品の単価はいくらですか？
Q-F2. ターゲットはどのくらいの情報リテラシーを持っていますか？
      → ①情報商材・広告に慣れた層 ②一般消費者に近い層 ③どちらとも言えない
Q-F3. 個別相談（Zoom・電話）でクロージングするリソースはありますか？
      → ①自分でできる ②セールス担当がいる ③難しい
Q-F4. 競合が多い市場ですか？
      → ①レッドオーシャン ②ニッチ ③わからない
Q-F5. コンセプト（素材）は圧倒的に強い自信がありますか？
      → ①ある ②まあまあ ③これから作る
```

## フェーズ2：バックエンド複数導線設計（核心）

フロントで取得したリストに対して、**意識の両極端**を狙った2〜3本目の導線を追加する。

### 思考の基本

```
フロントが狙う層：ミドル（中間）
    ↓
2本目：超顕在層向け（購買直前・専門知識あり）
3本目：超潜在層向け（まだ気づいていない・間接的アプローチ）
```

**ポイント：同じ商品・同じオファーページのまま、「教育・コミュニケーション」だけを変える。**
商品を増やす必要はない。

### 意識層別・導線設計の型

#### 超顕在層向け導線（技術・手法・実績で引き寄せる）
- 「○○を使って△△を実現する方法」という具体的タイトル
- 業界用語・専門手法を前面に出す
- 結果数字を具体的に示す（「1ヶ月で◯PV」「◯ヶ月で◯万円」）
- 3話型VSLよりも1本動画 or オートウェビナーが合いやすい

#### 超潜在層向け導線（理想の未来・感情・ライフスタイルで引き寄せる）
- 商品ジャンル名を一切使わない
- 「○○な人生を手に入れる方法」「△△から自由になる方法」
- 教育時間をたっぷり取る（30〜60分）
- 最後の最後に手段として商品を提示する
- 3話型VSLやウェビナー形式が有効

### 意識層の掘り下げワーク

ヒアリングで以下を確認し、両極端の切り口を提案する：

```
【バックエンド導線設計ワーク】

Q-B1. フロントのターゲット（ミドル層）が「すでに知っている専門的な内容」は何ですか？
      → これが超顕在層向け導線のフック候補

Q-B2. フロントのターゲットが「まだ意識していない根本的な悩み・欲求」は何ですか？
      → これが超潜在層向け導線の入口候補

Q-B3. ターゲットの属性（性別・年齢・職業）を教えてください
      → 属性が持つ「潜在的な関心」から潜在層向け導線を設計できます

Q-B4. 商品が解決する「手段」の上位にある「本質的な欲求」は何ですか？
      → 例：「ブログで稼ぐ」→「自由な働き方」→「家族との時間」
```

### 属性×潜在欲求マッピング

以下は「属性からアプローチできる潜在欲求」の参考マップ。

| 属性 | 潜在欲求の例（超潜在層向けフック） |
|---|---|
| 30〜40代男性会社員 | 「運の作り方」「仕事で評価される人の思考法」「副業で収入の柱を作る」 |
| 40〜50代女性 | 「潜在意識」「自己肯定感」「パートナーシップ」「自分らしい生き方」 |
| 20〜30代女性 | 「自分らしい働き方」「好きなことで生きる」「内面からの美しさ」 |
| 経営者・個人事業主 | 「組織文化」「経営判断の精度」「長期的なブランド構築」 |

## フェーズ0.5：リストを3ヶ月間生かし続ける仕組み

3ヶ月ROASを正確に計測するためには、**3ヶ月間リストが生きていること**が大前提。
以下の2つの仕組みを必ず組み込む。

---

### 仕組み①：メルマガ教育（2〜3日に1本・3ヶ月分）

**ルール：オファーは入れない。教育だけ。**

```
【メルマガ設計チェックリスト】

□ 配信頻度：2〜3日に1本（多すぎず少なすぎず）
□ 内容：純粋な教育コンテンツ（ノウハウ・考え方・事例）
□ オファー：メルマガ内には入れない（あくまで関係構築）
□ 本数：3ヶ月分（約30〜45本）
□ LINEと連動：LINE登録 → メルマガ誘導（ファンクスライン）
□ 開封率目安：30〜40%（この数字が出れば設計成功）
```

---

### 仕組み②：CS（カスタマーサポート）= ROASを1.5倍にする最大のレバー

**コスト：月60〜100万円　|　効果：数千万円規模の売上改善**

#### CS返信の3大カテゴリ

| カテゴリ | 内容 | 優先度 | 対応方針 |
|---|---|---|---|
| **技術系** | 「動画が見れない」「セミナーに参加できない」 | 低 | スルーOK（成約率への影響ほぼゼロ） |
| **商品質問系** | 「このサポートはありますか」「これは学べますか」 | 中 | テンプレ準備で対応 |
| **お気持ち系** | 感動した・こういう悩みがある・背中を押してほしい | **最高** | ここが成約に直結。最重要 |

#### お気持ち系テンプレ設計の手順

```
STEP 1：コアターゲットの属性を特定する
         例）「30代女性・主婦・副業初心者」

STEP 2：その属性が送ってくる「お気持ちメッセージ」を5パターン収集する
         例）
         ① 「感動しました！すごく刺さりました」（純粋な感動系）
         ② 「私もこういう状況で…」（共感・自己開示系）
         ③ 「本当にできるか不安です」（不安・躊躇系）
         ④ 「もっと詳しく聞きたいです」（前のめり系）
         ⑤ 「これは私には難しいかな…」（自己否定・後退系）

STEP 3：各パターンに「完璧な返信テンプレ」を作成する
         ・相手の感情を受け止める一言（共感）
         ・その感情を整理する一言（明確化）
         ・次のアクションへ自然に誘導する一言（CTA）

STEP 4：バイトスタッフに渡し、改変して返信させる
         → 時給1,000〜1,500円のパートで運用可能
```

#### 返信トリガーの設計（CSを動かすタイミング）

各導線の中に「必ず返信が来る仕掛け」を埋め込む。

| 導線タイプ | 返信トリガーのタイミング・方法 |
|---|---|
| **3話型VSL** | 各動画の終わりに「今日の気づき＋感想」を求める。潜在層向けは3択形式にすることで回答率UP |
| **オートウェビナー** | セミナー申込の1日前に「何を聞きたいですか？」をショート動画で投げる |
| **1本動画（個別相談）** | 受付期間中に1〜2回「こういう声が来ました、こう返しました」の実績アピールを挟む |

## フェーズ3：ライトロンチで検証してから広告投入

バックエンド導線の完成後、**いきなり広告費を使わず** ライトロンチで検証する。

```
【ライトロンチの手順】

STEP 1：コンセプトだけ作る（台本・スライド・完璧なコンテンツ不要）
         → セミナーまたはVSLの骨格だけ

STEP 2：既存リストに「興味ある人はこちら」と配信する

STEP 3：反応を見る
         → 良ければ本格制作・販売
         → 悪ければコンセプトを修正してから広告費を使う

STEP 4：ライトロンチの収益で広告費を賄い、フル展開する
```

**なぜライトロンチが重要か：**
反応がない導線に広告費を突っ込むリスクをゼロにするため。
「作ってから売る」ではなく「売れることを確認してから本格的に作る」。


## このOSが使える条件・制限

| 条件 | 最低ライン |
|---|---|
| 月間売上 | 200万円以上が望ましい（100万円以下だとリスト規模が不足しがち） |
| 商品の継続意志 | 3ヶ月以上運用し続ける前提。短期案件には不向き |
| リソース | CS運用のためのバイト雇用ができる体力 |
| 広告リスト | ある程度の規模（目安：月間新規獲得数十名以上） |

> **ビジネス系以外でも使える**
> 「意識層の両極攻め」はジャンルを選ばない。
> 40〜50代女性なら「潜在意識」から入る、30代男性なら「運の作り方」から入るなど、
> 属性が持つ潜在的関心を入口にすれば、どんな商品・ジャンルにも応用できる。
